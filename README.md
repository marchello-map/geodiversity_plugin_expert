# 🌍 Geodiversity Calculator v2.1 — Expert Edition

**Advanced geodiversity analysis with hybrid landform modelling (vector + geomorphon).**

This expert version extends the standard workflow with a **powerful, flexible geomorphology engine**, allowing users to combine **vector-based landform classification** and **DEM-derived geomorphon analysis** into a unified geodiversity framework.

---

## ✨ Features

- 🧮 Comprehensive geodiversity index calculation
- 🗺️ Supports vector and raster inputs
- 🔷 Multiple grid types:
  - Square
  - Hexagonal
  - Diamond
- ⚙️ Automated grid spacing suggestion based on study area size
- 🪨 **Advanced landform analysis with dual workflow support**
- 🌊 Integrated hydrography from DEM using Strahler stream order
- 📊 Optional normalized indices (0–1 scaling)
- 🎨 Automatic graduated symbology (Jenks classification)
- 📁 Output manifest generation for reproducibility
- ⚡ Designed for national-scale and other large datasets
